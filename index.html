PHY 504 Final Project

wordle.h provides header definitions for various aliased types, the class for the game, the derived class for the player, the ASCI color codings and the include files.

wordle.cpp contains the main function to start and play the game by instantizing a "player" who can access the game start/play functions, as well as output the results of the game.

game.cpp contains the class definitions for the class "game".

player.cpp contains the class definitions for the derived class "player".
